Twitch API Setup

For a reliable viewer counter you need a free Twitch Developer account.
1. Go to dev.twitch.tv/console
2. Click Register Your Application
3. Name: Stream Cockpit  |  OAuth URL: http://localhost  |  Type: Confidential
4. Copy Client ID and create a Client Secret

Without an API key a fallback method is used (less reliable)
